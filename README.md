# Visbran — Blog technique

Blog personnel sur l'administration système, l'infrastructure, le homelab et le DevOps.

🌐 **[visbran.fr](https://visbran.fr)**

## Stack

- **Framework** : [Astro](https://astro.build) (output statique)
- **Thème** : [Astro Pure](https://github.com/cworld1/astro-theme-pure) par [cworld1](https://github.com/cworld1)
- **Déploiement** : [Vercel](https://vercel.com)
- **Commentaires** : [Waline](https://waline.js.org)
- **Recherche** : [Pagefind](https://pagefind.app)

## Développement local

```bash
bun install
bun dev
```

## Crédits

Ce blog est basé sur le thème [Astro Pure](https://github.com/cworld1/astro-theme-pure), distribué sous licence [Apache 2.0](https://github.com/cworld1/astro-theme-pure/blob/main/LICENSE) par [cworld1](https://github.com/cworld1).

Les personnalisations incluent :
- Traduction complète en français (`src/i18n/`)
- Overrides de composants (`src/components/overrides/`)
- Configuration adaptée (`src/site.config.ts`)
